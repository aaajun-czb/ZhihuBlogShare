# AllWeather - 风险平价（Risk Parity）资产配置工具

## 项目简介
AllWeather 是一个基于**风险平价（Risk Parity）** 理论的资产配置计算工具，旨在通过量化方法实现资产组合的风险均衡分配。该工具支持自定义风险预算、计算资产收益率协方差矩阵（Ledoit-Wolf 收缩估计）、求解风险平价最优权重，并校验资产的实际风险贡献与目标风险预算的匹配度，最终输出组合年化波动率及资产权重分配结果。

## 项目结构
```
AllWeather/
├── config.json        # 配置文件（时间范围、资产列表、风险权重）
├── main.py            # 主程序（风险平价核心计算逻辑）
├── data_downloader.py # 资产数据下载脚本（基于akshare）
├── data/              # 资产净值数据存储目录（自动生成）
└── output/            # 输出目录（日志文件，自动生成）
```

### 文件说明
| 文件                | 功能说明                                                                 |
|---------------------|--------------------------------------------------------------------------|
| `config.json`       | 定义计算时间范围、资产列表（名称/代码/类型）、各资产目标风险权重          |
| `main.py`           | 核心逻辑：加载数据→计算协方差矩阵→风险平价优化→输出权重及风险贡献校验结果  |
| `data_downloader.py`| 自动下载配置文件中资产的历史净值数据，保存至`data/`目录（按代码命名CSV）|
| `data/`             | 存储资产净值CSV数据，文件名格式为「资产代码.csv」                         |
| `output/`           | 存储运行日志`risk_parity.log`，记录计算过程及最终结果                     |

## 环境依赖
需安装以下Python库：
```bash
pip install numpy pandas scipy scikit-learn akshare logging
```
> 注：`akshare_proxy_patch` 为自定义代理补丁（若无需代理可删除相关代码）。

## 快速开始

### 1. 配置文件修改
编辑`config.json`，按需调整时间范围、资产列表及风险权重：
```json
{
  "start_time": "2016-09-27",  // 计算起始日期
  "end_time": "2026-04-30",    // 计算结束日期
  "assets": [                  // 资产列表（支持OF/ETF/LOF类型）
    {
      "name": "大成中证红利指数A",    // 资产名称
      "code": "090010",             // 资产代码
      "type": "OF",                 // 资产类型（OF/ETF/LOF）
      "risk_weight": 0.125          // 目标风险权重（和建议为1）
    }
  ]
}
```

### 2. 下载资产数据
运行数据下载脚本，自动获取资产历史净值数据：
```bash
python data_downloader.py
```
> 下载成功后，`data/`目录下会生成对应资产代码的CSV文件，包含「净值日期」「累计净值」等字段。

### 3. 运行风险平价计算
执行主程序，完成风险平价权重求解：
```bash
python main.py
```

## 核心功能说明

### 1. 协方差矩阵计算
采用`LedoitWolf`收缩估计法计算资产日收益率的协方差矩阵，并默认年化（乘以252个交易日），提升协方差矩阵的稳定性。

### 2. 风险贡献计算
- **边际风险贡献（MRC）**：单个资产对组合波动率的边际影响
- **总风险贡献（TRC）**：单个资产占组合总风险的比例
- **风险预算**：自定义各资产的目标风险贡献比例（默认等风险）

### 3. 风险平价优化
通过`SLSQP`算法求解带约束的非线性优化问题，目标是最小化「实际风险贡献与目标风险预算」的差值平方和，约束条件：
- 权重和为1（满仓）
- 权重范围[0,1]（不做空、不加杠杆）

### 4. 结果校验
输出最优资产权重、实际风险贡献与目标风险预算的偏差、组合年化波动率，确保风险平价效果。

## 输出结果说明
运行完成后，日志文件`output/risk_parity.log`会记录以下核心信息：
1. **最优资产权重**：各资产的最终配置比例
2. **风险贡献校验**：实际风险贡献 vs 目标风险预算，及偏差值
3. **组合年化波动率**：最优权重下组合的年化波动率（风险水平）

示例输出：
```
1. 最优资产权重：
   大成中证红利指数A                          12.3450%
   博时标普500ETF（QDII）                     12.4560%
   
2. 风险贡献校验：
   大成中证红利指数A                          实际：12.5000% | 目标：12.5000% | 偏差：0.000000%
   
3. 组合年化波动率：8.9876%
```
