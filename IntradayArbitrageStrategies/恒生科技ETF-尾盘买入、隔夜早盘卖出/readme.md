# 恒生ETF（513130）日内交易策略（聚宽平台）

## 项目简介
本策略基于聚宽（JoinQuant）量化平台，针对恒生ETF（513130.XSHG）实现简单的日内交易逻辑：**开盘卖出持仓，收盘全额买入**，通过捕捉日内价差获取收益。

## 策略逻辑
### 1. 初始化设置（initialize）
- **基准设定**：以恒生ETF（513130.XSHG）为业绩基准
- **交易设置**：启用真实价格，开启动态复权
- **成本设定**：ETF交易佣金设为万分之0.1，无最低佣金、无印花税
- **定时任务**：
  - 开盘（`time='open'`）：执行卖出操作
  - 收盘（`time='close'`）：执行买入操作

### 2. 开盘卖出（sell_op）
- 检查持仓：仅当持有可卖仓位时才下单，避免无效委托警告
- 操作：卖出全部持仓（`order_target` 至0股）

### 3. 收盘买入（buy_op）
- 上市检查：通过 `get_security_info` 确认标的已上市，避免在上市前交易
- 操作：使用全部可用资金买入ETF（`order_value` 全额买入）

## 环境要求
- 平台：聚宽（JoinQuant）量化交易平台
- 依赖：聚宽官方函数库（jqdata）

## 使用说明
1. 登录聚宽平台，进入策略编辑页面
2. 将代码复制粘贴至策略编辑器
3. 如需调整标的、交易时间或成本，可修改 `initialize` 函数中的对应参数
4. 选择回测区间，运行策略回测；或根据平台规则进行模拟/实盘交易
