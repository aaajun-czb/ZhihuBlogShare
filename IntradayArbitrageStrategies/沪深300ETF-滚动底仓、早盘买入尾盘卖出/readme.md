# 沪深300ETF日内T+0策略回测代码

## 项目简介
本项目是基于聚宽（JoinQuant）平台开发的**沪深300ETF（510300.XSHG）日内T+0交易策略**。策略通过“底仓+日内波段”的方式，在保留底仓的基础上，利用每日可用资金进行日内买卖操作。

## 策略原理
### 核心逻辑
1. **底仓建立**：策略运行第一天，使用初始资金的 **49%** 建立底仓（剩余51%资金留作日内T仓操作）。
2. **日内T仓操作**：
   - **买入**：从第二天开始，每日开盘（09:30）用满当日可用资金买入ETF，买入数量不超过底仓股数（确保当日可全部卖出）。
   - **卖出**：每日收盘前（14:55）卖出当日买入的全部T仓，完成日内“T+0”循环。

## 回测说明
### 平台与标的
- **回测平台**：聚宽（JoinQuant）
- **交易标的**：沪深300ETF（代码：510300.XSHG）
- **业绩基准**：沪深300指数（代码：000300.XSHG）

### 关键参数
- **初始资金**：可在聚宽回测设置中自定义（建议不低于10万元，确保底仓与T仓操作空间）。
- **底仓比例**：初始资金的49%（`g.base_position_ratio = 0.49`）。
- **T仓资金**：用满当日可用资金（`g.t_funds_ratio = 1.0`）。
- **交易费用**：买入/卖出佣金万分之二，最低0.1元，无印花税。
