# 沪深300ETF融券做空策略（聚宽平台）

## 项目简介
本策略是基于聚宽（JoinQuant）量化平台开发的融资融券策略，通过融券卖出沪深300ETF（510300.XSHG）实现做空，并通过“早盘买券还券、尾盘融券开仓”的操作优化资金使用效率。

## 策略逻辑
### 1. 初始化设置
- 基准：沪深300指数（000300.XSHG）
- 启用真实价格交易
- 配置融资融券参数（融资利率8%、融券利率10%，保证金比例均为100%，资金使用效率90%）
- 设定定时任务：
  - 早盘开盘：买入ETF并归还前一日融券
  - 尾盘收盘：融券卖出开新仓
  - 收盘后：核查账户状态

### 2. 早盘操作（buy_and_refund_etf）
- 首日不操作，仅准备尾盘融券
- 非首日：按前一日融券数量买入ETF，买入后立即还券，释放保证金

### 3. 尾盘操作（short_sell_new_position）
- 根据可用保证金动态计算融券数量（取整至100股整数倍）
- 检查保证金充足后，融券卖出沪深300ETF
- 记录融券数量，供次日早盘还券

### 4. 收盘核查（check_account_status）
- 统计总资产、净资产
- 检查ETF多仓（理想值为0）、融券空仓数量
- 计算实际资金使用效率

## 环境要求
- 平台：聚宽（JoinQuant）量化交易平台
- 依赖：聚宽官方函数库（jqdata）

## 使用说明
1. 登录聚宽平台，进入策略编辑页面
2. 将本代码复制粘贴至策略编辑器
3. 如需调整参数，可修改`initialize`函数中的全局变量（如资金效率、两融利率等）
4. 选择回测区间/实盘交易，运行策略
